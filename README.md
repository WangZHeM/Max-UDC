# Max-UDC
Max-UDC实验代码
# READ ME

##### 本项目包含8个文件：

creatdata.py：数据生成文件

readdata.py：数据读取文件

MaxUDC.py：Max-UDC算法

OPT.py：暴力搜索算法

G_Greedy.py：Generalized greedy 算法

Greedy.py：Greedy算法

testfile.py: 测试代码

Data文件夹：存放实验数据

##### 使用步骤：

1.使用creatdata.py生成实验数据

2.运行Max-UDC.py、OPT.py、G-Greedy.py、Greedy.py进行实验
